What is markup?


  • To encode, to tag.
  • To make explicit (to a machine) what is implicit (to a person).
  • To add value by supplying multiple annotations.
  • To facilitate re-use of the same material.

XML Golden Rules

  • An element is represented by a start-tag, some optional content, and an end-tag.
  • An element may also have attributes, each consisting of a name and a value.
  • An XML document may also contain processing instructions comments or entity references.
  • An XML document must be well-formed and may be valid.
